# Tiny DNS

Tiny DNS is a simple DNS server written in Rust. It is a work in progress and it is not prepared for being used in production.

## Features

- [x] DNS over UDP.
- [ ] DNS over TCP.
- [x] Mirroring from other DNS servers.
- [ ] Caching.
- [x] Asynchronous.
- [ ] Log system.
- [x] Configuration.
